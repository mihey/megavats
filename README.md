# Megavats

Megavats is a Ruby wrapper of the Megafon VATS API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'megavats'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install megavats
